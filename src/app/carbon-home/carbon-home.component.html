What is Carbon?

Carbon is IBM’s open source design system for products and digital experiences.

With the IBM Design Language as its foundation, the system consists of working code, design tools and resources, human interface guidelines, and a vibrant community of contributors.

Carbon for Angular

Just use Carbon for Angular components if you won’t use Carbon for IBM.com Web components.

Mixing both may cause some issues, specially with scss conflicts and long compilation time.

Carbon for Web Components

There is no Carbon for IBM.com Angular components.

The Carbon for IBM.com team is focusing in Web Components even as a base for the Rect implementation.
